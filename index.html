Research

My focus is on AI, particularly in computer vision, deep learning, and image processing. I work on enhancing AI applications in cancer classification and video analysis, using advanced techniques like convolutional neural networks and lightweight models. Key projects include cancer image classification and real-time violence detection in videos.

Classification of Lung and Colon Cancer Histopathological Images Using Global Context Attention Based Convolutional Neural Network
Md. Al-Mamun Provath, Kaushik Deb, Pranab Kumar Dhar, Tetsuya Shimamura.

IEEE Access, 2023

Utilizing a global context attention-guided convolutional neural network, we significantly improved image and patient-level accuracy for cancer classification while reducing computational costs, making the model highly suitable for mobile deployment.

Classification of Lung and Colon Cancer Using Deep Learning Method
Md. Al-Mamun Provath, Kaushik Deb, Kang-Hyun Jo.

Springer Nature Singapore, 2023

By leveraging a computationally efficient CNN model and cyclic learning rate, we improved cancer detection accuracy to 97%, facilitating rapid identification of lung and colon cancers while maintaining computational efficiency.

XAI-Driven Lightweight Multiscale ConvLSTM Architecture for Video Violence Detection
Md. Al-Mamun Provath, Kaushik Deb, Kang-Hyun Jo.

IEEE, 2024

By introducing a novel deep neural network architecture with multiscale ConvLSTM and EvoNorm-S0, we achieved 92.40% accuracy in violence detection, with a processing time of 5 milliseconds, optimizing both accuracy and computational efficiency.
